
# AccrediFy Documentation Hub

Welcome to the central documentation for the AccrediFy platform. This directory contains detailed information about the project's architecture, purpose, data models, and development conventions. Each document is designed to provide clarity and guidance for developers, project managers, and future contributors.

- **[ROUTING_STRATEGY.md](ROUTING_STRATEGY.md)**: An explanation of how routing is handled on both the client-side (React) and server-side (Django API).
